# 이미지 생성 파이프라인

## 캐릭터 이미지

1. **얼굴 프리뷰** - Setup Wizard에서 seed 확정
2. **스탠딩 스프라이트** - Expression + Pose 조합
3. **CG 일러스트** - 레퍼런스 기반 수채화 스타일

## 프롬프트 전략

- 첫 번째 캐릭터: 스타일 기준 설정
- 추가 캐릭터: "Same art style as previous" 지시
- 동일 seed 사용으로 일관성 유지
